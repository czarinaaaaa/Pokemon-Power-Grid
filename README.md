**Pokémon Power Grid: Attack Stats Across Types and Traits**



**Description:**

 	This project explores Pokémon attack statistics using Python, Matplotlib and Seaborn. It compares attack values against defense and speed, shows distributions by type and highlights the strongest Dragon-type Pokémon. The goal is to demonstrate data visualization and storytelling with a fun dataset.



**Features:**

-Scatter plots (Attack vs Defense, Attack vs Speed)

\- Bubble Chart with HP as size

\- Violin plot showing attack distribution by type

\- Bar chart of top 5 Dragon Pokémon by attack



**Tools Used:**

\- Python

\- Pandas

\- Seaborn

\- Matplotlib



\## How to Run

1\. Clone the repository:

 	git clone https://github.com/czarinaaaaa/Pokemon-Power-Grid

2\. Navigate into the folder cd pokemon\_project

3\. Install dependencies:

 	pip install -r requirements.txt

4\. Run the script:

 	python dataset.py
